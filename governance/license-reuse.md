# License Reuse Policy

## Policy

TRAG is a fork of HKUDS/LightRAG. Reuse must preserve upstream notices, repository license terms, and attribution for third-party dependencies.

## Rules

- Do not remove upstream copyright, license, or attribution notices.
- Record new third-party code or assets in the relevant dependency manifest or documentation.
- Avoid copying external content into the repository unless its license permits reuse.

## Acceptance Criteria

- Required license files remain present.
- New reusable assets have source and license recorded.
- Ambiguous license cases are blocked until reviewed.
